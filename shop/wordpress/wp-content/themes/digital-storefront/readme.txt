== Description ==

Digital Storefront gives you a choice to create your ecommerce store with ease. It is a brilliant theme suitable to create website for smart Home products and accessories, gadget store, Jewellery Market, fashion shop, ecomerce store, secrets santa, gift shops, watch store, decorative stores, AR/VR Headsets, home appliances shop, grocery store, Hardware, Electric Scooters, 3D Assets Stores, online support, and other such related websites. The theme is professional, clean, elegant and retina-ready in look. It is SEO-friendly theme which will help your website rank soon on major search engines like Google. It is translation-ready and supports RTL layout. It is based on bootstrap framework that makes it very handy to use. It is optimized for speed hence gives faster page load times. It is written with clean and secure codes. The responsive layout makes it mobile-friendly that looks perfect on any device. It is user-friendly too with many options for customization. It is compatible with major WordPress plugins like Contact form 7, WooCommerce etc. Also, it is well sanitized as per WordPress standards. == Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Digital Storefront in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
